소통픽

엑셀·한글 문서의 수신자 명단을 붙여넣기만 하면
소통메신저에서 자동으로 검색하고 선택합니다.

⬇  무료 다운로드 (.exe) 사용 방법 보기 →

Windows 전용  ·  무설치  ·  더블클릭으로 바로 실행

주요 기능
반복 작업, 이제 맡기세요

수십·수백 명의 수신자를 일일이 검색하고 클릭하는 작업을 자동으로 처리합니다.

📊
엑셀 파일
.xlsx 파일에서 소속기관과 이름을 자동으로 읽어옵니다.
📄
한글 HWP
.hwp · .hwpx 파일도 직접 열어 명단을 추출합니다.
📋
복사·붙여넣기
Ctrl+C로 복사한 표를 바로 붙여넣기 해도 됩니다.
🔍
오타 자동 보정
기관명에 오타가 있어도 충북 기관 DB 기반으로 자동 매칭합니다.
사용 방법
3단계로 끝납니다

처음 사용하시는 분도 아래 순서대로 따라 하시면 됩니다.

  • 1
    소통메신저 발송 창 열기
    오른쪽 위 편지 아이콘(✉) → 쪽지 작성 또는 대화하기 → [사용자 선택] → 전체조직 탭 클릭
  • 2
    명단 입력 — 두 가지 방법 중 선택
    방법 ① 복사·붙여넣기 — 엑셀에서 소속기관(A열)·이름(B열) 선택 후 Ctrl+C, 입력창에 Ctrl+V
    방법 ② 파일 직접 열기 — [엑셀 파일 열기] 또는 [HWP 파일 열기] 버튼 클릭
    A열 — 소속기관B열 — 이름
    충주중학교홍길동
    청주고등학교김철수
    충청북도교육청이영희
  • 3
    [명단 추출 →] 클릭
    소속기관과 이름이 자동 인식되어 목록에 표시됩니다. 목록을 더블클릭하면 항목을 직접 수정할 수 있습니다.
ℹ
최초 1회만 설정하면 됩니다. 소통메신저 창 위치가 바뀌지 않는 한 재설정 불필요합니다. 자동 선택은 검색 → 결과 클릭 → 선택 버튼 클릭의 3단계로 동작합니다.
  • 1
    STEP 1 — 검색 입력창 위치
    [📍 위치 설정] → [캡처 시작] 클릭 → 3초 안에 소통메신저 검색 입력칸 위로 마우스 이동
  • 2
    STEP 2 — 결과 첫 번째 항목 위치
    임의 이름 검색 후 결과 목록의 첫 번째 줄이 보이는 상태에서 [📍 위치 설정] → [캡처 시작] → 3초 안에 마우스 이동
  • 3
    STEP 3 — 사용자 선택 버튼 위치
    결과 목록이 보이는 상태에서 [📍 위치 설정] → [캡처 시작] → 3초 안에 소통메신저의 [사용자 선택] 또는 [추가] 버튼 위로 마우스 이동
  • 4
    [✅ 설정 저장] 클릭
    반드시 저장해야 다음 실행 시에도 설정이 유지됩니다.
  • 1
    [▶ 자동 선택 시작] 클릭
    소통메신저 사용자 선택 창이 열려있는 상태에서 시작하세요.
  • 2
    자동 진행 — 마우스를 움직이지 마세요
    프로그램이 이름을 검색하고 자동으로 선택합니다. 진행 상황이 로그에 실시간 표시됩니다.
  • 3
    완료 — 결과 확인
    ✓ 성공 / ✗ 실패 건수가 요약됩니다. 빨간색 항목은 검색 결과가 없는 경우입니다.
⚠
긴급 중지: 마우스를 화면 왼쪽 위 모서리(0, 0)로 빠르게 이동하거나 [■ 중지] 버튼을 클릭하세요.
FAQ
자주 묻는 질문
다운로드 후 실행이 안 돼요. Windows 경고가 뜹니다.
추가 정보를 클릭한 뒤 실행을 선택하세요. 소규모 배포 프로그램은 코드 서명 인증서가 없어 SmartScreen 경고가 표시될 수 있습니다. 소스 코드는 GitHub에서 누구나 확인할 수 있습니다.
소속기관이 '소속없음'으로 표시돼요.
DB에 없는 기관명이거나 오타가 심한 경우입니다. 추출된 목록에서 해당 항목을 더블클릭하면 직접 수정할 수 있습니다.
프로그램이 엉뚱한 위치를 클릭해요.
소통메신저 창 위치가 바뀌었을 수 있습니다. [2. 위치 설정] 탭에서 위치를 다시 설정하고 저장하세요.
동명이인이 있을 때는 어떻게 하나요?
[2. 위치 설정] 탭에서 수동 확인 모드를 체크하세요. 각 이름 검색 후 소통메신저에서 직접 선택하고 [▶▶ 계속] 버튼을 눌러 진행합니다.
HWP 파일이 열리지 않아요.
한/글이 설치되지 않은 경우 일부 형식이 지원되지 않을 수 있습니다. 한글 문서에서 표를 Ctrl+C로 복사한 뒤 입력창에 붙여넣으세요.
속도를 조절할 수 있나요?
[2. 위치 설정] 탭의 검색 후 대기 시간에서 조절하세요. 기본값은 0.5초이며, 인터넷이 느린 환경에서는 1.5~2초로 늘려보세요.
