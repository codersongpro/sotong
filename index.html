주요 기능

반복 작업, 이제 맡기세요

수십 명, 수백 명의 수신자를 일일이 검색하고 클릭하는 작업을 자동으로 처리합니다.

📊
엑셀 파일

한글 HWP
.hwp / .hwpx 파일도 직접 열어 명단을 추출합니다
📋
복사·붙여넣기
표를 Ctrl+C로 복사 후 바로 붙여넣기로도 사용 가능
🔍
오타 자동 보정
소속기관명에 오타가 있어도 DB 기반으로 자동 매칭합니다

사용 방법

3단계로 끝납니다

처음 사용하시는 분도 아래 순서대로 따라 하시면 됩니다.

  • 1
    소통메신저에서 발송 창 열기 소통메신저 오른쪽 위 편지 아이콘(✉) 클릭 → '쪽지 작성' 또는 '대화하기' 선택
  • 2
    [사용자 선택] → [전체조직] 탭 열기 메시지 창에서 [사용자 선택] 버튼 클릭 후 상단 탭에서 전체조직을 선택하세요.
  • 3
    명단 붙여넣기 또는 파일 열기 엑셀에서 소속기관(A열)과 이름(B열)을 함께 선택해 복사(Ctrl+C)한 뒤 붙여넣기 하세요. 또는 엑셀·HWP 파일을 직접 열 수 있습니다.
    A열 — 소속기관 B열 — 이름
    충주중학교 홍길동
    청주고등학교 김철수
    충청북도교육청 이영희
  • 4
    [명단 추출 →] 클릭 소속기관과 이름이 자동으로 인식되어 아래 목록에 표시됩니다. 오타가 있어도 자동 보정됩니다.
ℹ
최초 1회만 설정하면 됩니다.

소통메신저 창 위치가 바뀌지 않는 한 재설정 불필요합니다.

  • 1
    STEP 1 — 검색 입력창 위치 설정 [📍 위치 설정] 클릭 → 3초 안에 소통메신저 검색 입력칸 위로 마우스를 이동하세요.
  • 2
    STEP 2 — 결과 첫 번째 항목 위치 설정 임의의 이름을 검색 후 결과 첫 번째 줄 위에서 [📍 위치 설정]을 클릭하세요.
  • 3
    STEP 3 — 결과 없음 감지 설정 (선택) 존재하지 않는 이름 검색 후 결과가 빈 상태에서 [📍 픽셀 캡처] 클릭. 이후 검색 결과가 없으면 자동으로 ✗ 처리합니다.
  • 4
    [✅ 설정 저장] 클릭 반드시 저장해야 다음에도 설정이 유지됩니다.
  • 1
    [▶ 자동 선택 시작] 클릭 소통메신저 사용자 선택 창이 열려있는 상태에서 시작하세요.
  • 2
    자동 진행 — 마우스를 움직이지 마세요 프로그램이 이름을 검색하고 자동으로 선택합니다. 진행 상황이 로그에 실시간 표시됩니다.
  • 3
    완료 — 결과 확인 ✓ 성공 / ✗ 실패 건수가 요약됩니다. 빨간색으로 표시된 항목은 검색 결과가 없는 경우입니다.
⚠
긴급 중지 방법

마우스를 화면 왼쪽 위 모서리(0,0)로 빠르게 이동하거나 [■ 중지] 버튼을 클릭하세요.

FAQ

자주 묻는 질문

궁금한 점이 있으면 아래에서 확인하세요.

다운로드 후 실행이 안 돼요. Windows 경고가 뜹니다.
추가 정보를 클릭한 뒤 실행을 선택하세요.
소규모 배포 프로그램은 코드 서명 인증서가 없어 Windows SmartScreen 경고가 표시될 수 있습니다. 소스 코드는 GitHub에서 누구나 확인할 수 있습니다.
소속기관이 '소속없음'으로 표시돼요.
DB에 없는 기관명이거나 오타가 심한 경우입니다. 입력 시 소속기관명을 정확하게 입력하거나, 소속기관 없이 이름만 입력해도 검색이 가능합니다.
프로그램이 엉뚱한 위치를 클릭해요.
소통메신저 창 위치가 바뀌었을 수 있습니다. [2. 위치 설정] 탭에서 위치를 다시 설정하고 저장하세요.
동명이인이 있을 때는 어떻게 하나요?
[2. 위치 설정] 탭에서 수동 확인 모드를 체크하세요. 각 이름 검색 후 소통메신저에서 직접 더블클릭하고, [▶▶ 계속] 버튼을 눌러 진행합니다.
HWP 파일이 열리지 않아요.
한/글 프로그램이 설치되지 않은 경우 일부 형식이 지원되지 않을 수 있습니다. 한글 문서를 직접 열어 표를 Ctrl+C로 복사한 뒤 입력창에 붙여넣으세요.
속도를 조절할 수 있나요?
[2. 위치 설정] 탭의 검색 후 대기 시간에서 조절하세요. 기본값은 0.5초이며, 인터넷이 느린 환경에서는 1.5~2초로 늘려보세요.
