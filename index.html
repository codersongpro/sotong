엑셀·한글 명단을 붙여 넣으면 소통메신저에서 수백 명을 자동 검색·선택합니다.

⬇ 프로그램 다운로드 (.exe)
Windows 전용 · 무설치 · 최신 버전 자동 연결

🚀Python 없어도 됩니다 — 5분 만에 시작하기

✅
이 프로그램은 설치 없이 바로 실행되는 .exe 파일로 제공됩니다.
Python을 설치하지 않아도 됩니다. 아래 순서대로 따라 하세요.
  1. 프로그램 다운로드 위의 프로그램 다운로드 버튼을 클릭하면 GitHub 릴리즈 페이지로 이동합니다. ChungbukMessenger_AutoSelect.exe 파일을 클릭해 저장하세요.
  2. 보안 경고 허용 Windows Defender SmartScreen이 뜨면 "추가 정보" → "실행"을 클릭하세요. (소규모 배포 프로그램은 처음 실행 시 경고가 표시될 수 있습니다.)
  3. 프로그램 실행 다운로드된 .exe 파일을 더블클릭하면 바로 실행됩니다. 별도 설치 과정 없음.
  4. 소통메신저 로그인 후 준비 충북 소통메신저에 로그인하고 사용자 선택 창을 열어 두세요. 이 프로그램 창과 나란히 배치하면 편리합니다.
  5. 명단 입력 → 위치 설정(최초 1회) → 자동 선택 시작 아래 사용 방법을 참고하세요.

💡이 프로그램이 하는 일

충청북도교육청 소통메신저에서 수신자를 추가할 때, 수십~수백 명의 이름을 일일이 검색하고 클릭하는 반복 작업을 자동화합니다.

📊

.xlsx 파일에서 소속기관·이름을 자동 추출
📄

.hwp / .hwpx 파일 직접 읽기 가능
📋

문서에서 Ctrl+C 복사 후 바로 붙여넣기
🔍
오타 자동 보정
소속기관명 오타도 DB 기반 자동 매칭
ℹ
소속기관 DB 내장: 충청북도교육청 직속기관 12개, 고등학교 84개, 중학교 132개, 초등학교 257개, 특수·각종학교 9개의 명칭이 내장되어 있어 오타가 있어도 자동으로 정확한 명칭으로 변환합니다.

📖사용 방법

① 탭 1 — 명단 입력

소속기관과 이름이 들어있는 명단을 세 가지 방법으로 입력할 수 있습니다.

📂
방법 A: 엑셀 파일
[엑셀 파일 열기] → .xlsx 선택 → [명단 추출 →]
📃
방법 B: HWP 파일
[HWP 파일 열기] → .hwp 선택 → [명단 추출 →]
📋
방법 C: 복사·붙여넣기
표를 Ctrl+C 복사 → 입력창에 Ctrl+V → [명단 추출 →]

ℹ
인식 가능한 형식: 소속기관  이름 (탭 구분) 또는 줄마다 소속↔이름 교대. 소속기관명의 오타도 자동 보정됩니다.
② 탭 2 — 위치 설정 (최초 1회만)

소통메신저 창의 어느 위치를 클릭해야 하는지 알려주는 과정입니다. 한 번만 설정하면 이후 자동으로 기억합니다.

  1. STEP 1 — 검색 입력창 위치 소통메신저 검색창 위에서 [📍 위치 설정]을 클릭해 3초 안에 검색칸 위에 마우스를 올리세요.
  2. STEP 2 — 결과 첫 번째 위치 임의의 이름 검색 후 결과 첫 번째 줄 위에서 [📍 위치 설정]을 클릭하세요.
  3. STEP 3 — 결과 없음 감지 (선택) 존재하지 않는 이름 검색 후 결과가 빈 상태에서 [📍 픽셀 캡처]를 클릭하세요. 이후 결과 없는 경우 자동으로 ✗ 처리합니다.
  4. [✅ 설정 저장] 클릭 반드시 저장해야 다음에도 설정이 유지됩니다.
③ 탭 3 — 자동 선택 시작
  1. 소통메신저 준비 [사용자 선택] → [전체조직] 탭 열기
  2. [▶ 자동 선택 시작] 클릭 프로그램이 자동으로 검색·더블클릭을 반복합니다. 마우스를 건드리지 마세요.
  3. 진행 상황 확인 로그창에 ✓(성공) / ✗(결과 없음)이 표시됩니다. 완료 시 결과가 요약됩니다.
⚠
긴급 중지: 마우스를 화면 왼쪽 위 모서리(0,0)로 빠르게 이동하거나 [■ 중지] 버튼을 클릭하세요.

🐍Python 개발자용 — 소스 코드 실행

소스 코드를 직접 실행하거나 수정하고 싶은 경우, Python 환경에서 아래 방법을 사용하세요.

⚠
일반 사용자는 위의 .exe 파일 다운로드를 사용하세요. Python 설치가 필요 없습니다.

1. Python 설치 (미설치 시)

python.org/downloads에서 Python 3.10 이상을 설치하세요. 설치 시 "Add Python to PATH"에 반드시 체크하세요.

2. 필수 패키지 설치

pip install pyautogui pyperclip openpyxl

3. 실행

python main.py

4. 소스 코드 / 기여

❓자주 묻는 질문

소속기관 이름이 없거나 '소속없음'으로 처리돼요.
DB에 등록되지 않은 소속기관명이거나 오타가 심한 경우입니다. 입력창에서 직접 소속기관[탭]이름 형식으로 수정 후 [명단 추출 →]을 다시 클릭하세요.
프로그램이 엉뚱한 곳을 클릭해요.
소통메신저 창 위치가 바뀌었을 수 있습니다. [2. 위치 설정] 탭에서 위치를 다시 설정하고 [✅ 설정 저장]을 눌러주세요.
Windows SmartScreen 경고가 뜨면 어떻게 하나요?
추가 정보를 클릭 → 실행을 선택하세요. 이 프로그램은 소규모 배포 .exe로, 코드 서명 인증서가 없어 경고가 표시될 수 있으나 안전한 파일입니다. 소스 코드는 GitHub에서 직접 확인할 수 있습니다.
동명이인이 있을 때 어떻게 하나요?
[2. 위치 설정] 탭에서 수동 확인 모드를 체크하세요. 검색 후 [▶▶ 계속] 버튼이 활성화되면, 소통메신저 결과에서 직접 더블클릭한 뒤 [계속]을 누릅니다.
HWP 파일이 열리지 않아요.
한/글 프로그램이 설치되어 있지 않으면 일부 .hwp 파일이 열리지 않을 수 있습니다. 한글 문서를 직접 열어 해당 표를 Ctrl+C로 복사한 뒤, 프로그램 입력창에 Ctrl+V로 붙여넣어 사용하세요.
속도가 너무 빠르거나 오류가 자주 발생해요.
[2. 위치 설정] 탭의 검색 후 대기 시간을 늘려보세요. 인터넷이 느린 환경에서는 1.5~2.5초, 빠른 환경에서는 0.8~1.2초가 적당합니다.
